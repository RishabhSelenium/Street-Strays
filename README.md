# Street Strays

Street Strays is a single-file browser game inspired by flocking/herding mechanics. The player guides a helper dog to move packs of street strays into a shelter before nightfall while navigating city hazards like rubble, flooded drains, traffic, and catchers.

## Play

Open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server 4175
```

Then open:

```text
http://localhost:4175/index.html
```

## Current Build

- Single-file HTML/CSS/JavaScript game.
- 20-level Street Strays campaign.
- Mouse and touch controls.
- Local storage progress support.
- Testing flag currently unlocks all levels for playtesting.
